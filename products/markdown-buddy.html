Markdown Buddy

En bref : Aperçu Markdown instantané pour développeurs — depuis l'éditeur ou le Finder.

Plateformes : macOS

Contexte

Les développeurs rédigent README, documentation et notes en Markdown, mais prévisualiser le rendu impose souvent de quitter son workflow ou d'ouvrir un navigateur. Markdown Buddy intègre l'aperçu directement dans l'environnement de travail.

Approche produit

Extension Source Editor pour Xcode (⌃⇧M) + viewer autonome. Glisser-déposer, onglets multiples, auto-refresh à la sauvegarde. Style GitHub, dark mode natif. App Sandbox, aucune donnée collectée.

Rôle

Vision produit · UX · Développement macOS · Extension Xcode (Source Editor) · Rendu Markdown

Fonctionnalités clés

  • extension Xcode : raccourci ⌃⇧M pour prévisualiser le fichier ouvert
  • viewer autonome : ouvrir des .md depuis le Finder ou par glisser-déposer
  • onglets multiples et auto-refresh à la sauvegarde
  • Markdown complet : titres, listes, tableaux, code, citations, images
  • dark mode natif et style GitHub
